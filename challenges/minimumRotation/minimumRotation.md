### Minimum rotations required to get the same String 

Given a string, we need to find the minimum number of rotations required to get the same string. 
In this case, we will only consider Left rotations.

Example:
```
Input : s = “geeks”
Output : 5

Input : s = “aaaa”
Output :1 
```
