## Letter Capitalize

Write a function that takes a string and capitalizes the first letter of every word.  Words will be separated by only one space.

#### Example:

```javascript
f("hello world")
// "Hello World"

f("101 Dalmatians is a great movie")
// "101 Dalamatians Is A Great Movie"
```
